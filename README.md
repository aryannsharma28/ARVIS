## ğŸ” Project Description

The **Serverless Online Ticketing Platform** is a modern web application that enables users to book tickets for various transportation services and events through a unified interface. The system is designed using a **serverless deployment model**, allowing it to scale automatically based on demand without manual infrastructure management.

Developed using **PHP**, the application runs in containers deployed on **Google Cloud Run**. Secure online payments are processed through the **Cashfree payment gateway**, while **MySQL** is used to store booking, user, and transaction data. **Docker** ensures consistent environments across development and production.
